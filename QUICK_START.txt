================================================================================
DATA STORAGE
================================================================================

All data is stored in browser's localStorage:
  • User accounts
  • Bookings
  • Cart items
  • Session information

To clear all data (reset everything):
  1. Open browser DevTools (F12)
  2. Go to Application → Local Storage
  3. Find eventhub_* entries
  4. Delete them
  5. Refresh page

Or run in console:
  Object.keys(localStorage).forEach(key => {
    if (key.startsWith('eventhub_')) {
      localStorage.removeItem(key);
    }
  });

================================================================================
TROUBLESHOOTING
================================================================================

Q: Cart not saving?
A: Make sure localStorage is enabled in browser
   Try clearing cache and refreshing

Q: Payment not processing?
A: Use exact test card: 4242 4242 4242 4242
   Check browser console for any errors
   Wait 2 seconds for processing

Q: Demo account not working?
A: Check email is exact (case-sensitive)
   Verify password is correct
   Try signing up with same role instead

Q: Booking not showing?
A: Go to "My Bookings" page
   Make sure you're logged in as customer
   Check browser's localStorage is enabled

Q: Vendor dashboard not showing?
A: Sign out and sign back in
   Use correct role account (venue or provider)
   Check profile menu for dashboard link
